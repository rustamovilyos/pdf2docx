## Run Locally

Clone the project

```bash
  git clone https://github.com/rustamovilyos/pdf2docx
```

Go to the project directory

```bash
  cd pdf2docx
```

Install requirements

```bash
  pip install -r requirements.txt
```

Run project

```bash
  python main.py
```
